How To Use

   This visualization tool shows the steps taken during the incremental Delaunay Triangulation algorithm. The insertion order follow the order that the points were drawn on the canvas. To use this tool click on the canvas to add points, once you have the points press "Submit". Then you can either "Run" the incremental program at the desired speed or you can switch to Manual mode and press "Next" to go through each step. If you want to pause and switch from Automatic to Manual at any point you can. To input new points simply "Clear" the canvas and start again. Note: if you put points too close together it may not run, if this happens just clear the canvas and put in new points.

Delaunay Triangulations

   A Delaunay Triangulation is a specific type of triangulation that is used all the time in Geographic Information Systems (GIS). They maximizes the minimum angle of the triangulation and thus gives us a triangulation where each triangle has angles that are similar to each other. This is especially useful in GIS since it allows us our interpolations of data within each of these triangles to be likely more accurate than if we were using a triangulation that didn't have this property.

Incremental Construction Algorithm

   One of the popular ways to create a Delaunay Triangulation is through incremental construction. The way this works is first a super triangle with dummy points is created to encompass each of the points of interest. Once this is done, the algorithm picks a point to insert, finds which triangle this point is in and retriangulates it with the vertices of that triangle. After this retriangulation is done the algorithm then preforms an "incircle" check for each of the newly created triangles. When doing the incircle check we draw the circumcircle for each of the triangles and if any other points lie in that circle then we must flip the edge that lies between the newly inserted point and the point that falls within the circle. We then preform incircle check on the modified triangles and preform edge flips until no triangles violate the incircle property. Then we pick our next point and repeat the insertion process. Once we have all points inserted then we have a Delaunay Triangulation. This visualisation program allows you to follow along with the incircle checks and edge flips of the triangulation.
